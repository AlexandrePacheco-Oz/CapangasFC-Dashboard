# Capangas FC - Dashboard de Mensalidades ⚽

Um dashboard moderno e responsivo para controle de mensalidades de membros do **Capangas Futebol Clube**.

## ✨ Funcionalidades

- **Design Premium**: Interface baseada em *Glassmorphism* com modo escuro e identidade visual do clube (Azul e Dourado).
- **Dashboard de Métricas**: Visualização em tempo real do total arrecadado, número de pagamentos confirmados e pendentes.
- **Busca Rápida**: Filtro dinâmico para encontrar membros na lista instantaneamente.
- **Marca d'água Dinâmica**: Fundo personalizado com o escudo do clube.
- **Persistência Local**: Dados salvos automaticamente no navegador (`localStorage`).
- **Acesso Administrativo**: Controle de pagamentos protegido por login simples.

## 🚀 Como usar

1. Clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Para marcar pagamentos, utilize o login padrão:
   - **Usuário**: `admin`
   - **Senha**: `123`

## 🛠️ Tecnologias

- **HTML5** (Estrutura semântica)
- **CSS3** (Variáveis, Flexbox, Grids, Glassmorphism, Google Fonts)
- **JavaScript** (Lógica de renderização, filtros e armazenamento local)
